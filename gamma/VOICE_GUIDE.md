# 🎤 语音识别使用指南

技术思维处理器现在支持**实时语音识别**功能，让您可以通过语音输入技术问题，系统会自动转换为文本并进行AI分析。

## 🚀 快速开始

### 1. 浏览器要求
- ✅ **Chrome** 25+ (推荐)
- ✅ **Edge** 79+
- ✅ **Safari** 14.1+
- ✅ **Firefox** (部分支持)

### 2. 权限设置
首次使用时，浏览器会要求麦克风权限：
1. 点击地址栏的 🎤 图标
2. 选择"允许"
3. 刷新页面

### 3. 基本操作
```
1. 初始化处理器 → 配置API密钥 → 保存启动
2. 返回主页 → 点击"开始语音识别"
3. 清晰说出技术问题
4. 观察实时分析结果
```

## 🎯 使用技巧

### 语音输入最佳实践

#### 📢 说话技巧
- **语速适中**：不要太快或太慢
- **吐字清晰**：特别是技术术语
- **环境安静**：减少背景噪音
- **距离适当**：麦克风距离嘴巴15-30cm

#### 🔤 技术词汇
语音识别对技术术语的效果：

**效果良好：**
```
✅ "什么是分布式系统"
✅ "数据库查询优化"  
✅ "Python装饰器"
✅ "机器学习算法"
✅ "网络协议"
```

**需要注意：**
```
⚠️ "MySQL" → 可能识别为"my sequel"
⚠️ "JavaScript" → 建议说"JS"
⚠️ "API" → 建议说"接口"
⚠️ "CSS" → 建议说"样式表"
```

## 🎛️ 功能详解

### 识别模式

#### 连续识别模式
- **特点**：持续监听，自动处理停顿
- **适用**：长句子、复杂问题
- **示例**：
  ```
  "请解释一下什么是微服务架构，
   以及它与单体架构的区别，
   还有实施时需要注意哪些问题"
  ```

#### 实时识别模式  
- **特点**：边说边显示，即时反馈
- **适用**：短句、快速输入
- **示例**：
  ```
  "Redis"   → 立即显示
  "缓存"     → 实时追加
  "原理"     → 继续追加
  ```

### 状态指示器

#### 🟢 正在监听
```
• 绿色波浪动画
• 正常录音状态
• 可以开始说话
```

#### 🔴 识别中
```  
• 红色脉冲动画
• 正在处理语音
• 稍等片刻
```

#### ⚡ 置信度显示
```
90%+ → 识别准确，可信度高
70-90% → 识别良好，请确认
<70% → 建议重新说一遍
```

## 🔧 问题排查

### 常见问题

#### ❌ "无法访问麦克风"
**解决方案：**
1. 检查浏览器权限设置
2. 确认麦克风硬件正常
3. 重启浏览器

#### ❌ "没有检测到语音输入"
**解决方案：**
1. 检查麦克风音量
2. 靠近麦克风说话
3. 减少环境噪音

#### ❌ "识别结果不准确"
**解决方案：**
1. 吐字更清晰
2. 降低语速
3. 使用简单词汇

#### ❌ "网络错误"
**解决方案：**
1. 检查网络连接
2. 刷新页面重试
3. 切换网络环境
